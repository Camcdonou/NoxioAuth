This map editor is what I used to make maps for the game. It's not perfect and may have issues.
The maps are plain text and can be hand edited if needed. (I have to do that occasionally haha)


Additionally, if you would like to add new textures, materials, models, or glsl shaders to use in a map you will have to contact me directly.
Twitter and Discord are fine. Compiling in new stuff takes a bit of effort though so please have all your assets prepared thoroughly.
I'm not going to make a production build just to add a meme jpeg.

Main Hotkeys
g - Toggle Grid
c - Draw Collision
d - Draw Doodads
s - Draw Spawns
r - Rotate

Collision Hotkeys
rmb - Select Point
lmb - Move Point
ctrl+rmb - Create Collision
ctrl+lmb - Add Point
del - Delete Point
ctrl+del - Delete Collision

Doodad Hotkeys
rmb - Select Doodad
lmb - Move Doodad
ctrl+rmb - Create Doodad
ctrl+lmb - Rotate Doodad
del - Delete Doodad
pgup/pgdwn - Raise / Lower

Spawn Hotkeys
rmb - Select Spawn
lmb - Move Spawn
ctrl+rmb - Create Spawn
ctrl+lmb - Rotate Spawn
